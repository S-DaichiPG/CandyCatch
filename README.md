## 基本操作
1. 初期画面で0か1を入力し難易度を選択(決定はEnter)　※難易度が上がるとアイテムの落下速度が速くなる
2. 矢印キー（← →）で左右に移動
3. キャンディをとるとポイントが加算される
5. 3回カボチャに接触するとゲーム終了


※ESCキーで途中終了も可能
